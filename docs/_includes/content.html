About Check Yo Self

Check Yo Self has a simple purpose: to check your markdown for grammatical errors and misspellings.

Features

  • Responsive design
  • Mobile friendly
  • You can pick up right were you left off with an account.

For more information, including setup and configuration instructions, head over to The Docs.

For bug reports and feature requests, pop on over to GitHub and leave an issue.
